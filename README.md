# Muscles

**Kalória követő, edző-kliens összekötő alkalmazás. [Próbáld ki!](http://34.22.242.178:3000/)**

- Étkezés nyomon követés
- Edző-kliens össze kötés

- Súly nyomon követés
- Visszamenőleges adatok
- Egyéni célok
